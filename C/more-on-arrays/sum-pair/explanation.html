Introduction

Write a program that, given an array A[] of n numbers and another number x, determines whether or not there exist two elements in A[] whose sum is exactly x.


Talking Simply

So we need to write a program in which we will get or use a number like 3 and an array like {1, 3, 2} so our plan is to find two numbers in the array whose sum is 3 so the ideal output for this situation will be 2 and 3 because 2 + 3 = 5 .

Some More Examples

Input: arr[] = {0, -1, 2, -3, 1}
sum = -2

Output: -3, 1

Explanation:
If we calculate the sum of the output,
1 + (-3) = -2

Input: arr[] = {1, -2, 1, 0, 5}
sum = 0

Output: -1

Explanation:
No valid pair exists.
